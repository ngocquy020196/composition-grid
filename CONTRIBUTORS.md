## How to Contribute

We welcome contributions! Here's how you can help:

1. **🐛 Report Bugs** — [Open an issue](https://github.com/ngocquy020196/Composition-Grid/issues/new)
2. **💡 Suggest Features** — Share your ideas via issues
3. **🔧 Submit PRs** — Fork, code, and submit a pull request
4. **📖 Improve Docs** — Fix typos, add examples, clarify instructions
5. **⭐ Star the Repo** — Show your support!

### Contribution Guidelines

- Fork the repository and create your branch from `main`
- Follow existing code style and conventions
- Test your changes before submitting
- Write clear commit messages
- Update documentation if needed
